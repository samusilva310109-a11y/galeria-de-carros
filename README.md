# galeria-de-carros
## Descrição
A galeria tem por objetivo praticar a disposição de imagens dentro de uma página web com a tag "img" e o hyperlink "a" utilizado para redirecionarr o usuário a outra página. A atividade é tema livre a gosto do aluno, portanto foi escolhido o tema de carros antigos.

**CSS:** com foco na utilização da tecnologia display flex
